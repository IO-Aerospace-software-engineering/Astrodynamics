\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classIO_1_1Astrodynamics_1_1Body_1_1CelestialBody}{Celestial\+Body}} class represents a celestial body in astrodynamics calculations. It inherits from \mbox{\hyperlink{classIO_1_1Astrodynamics_1_1Body_1_1CelestialItem}{IO\+::\+Astrodynamics\+::\+Body\+::\+Celestial\+Item}}. 

A celestial body is characterized by its sphere of influence, hill sphere, body-\/fixed frame, and certain J values. It provides methods to obtain various properties such as the relative state vector, sphere of influence, hill sphere, body-\/fixed frame, radius, flattening, angular velocity, and rotation period.

It also provides static methods to check if a given celestial body is a sun, planet, asteroid, moon, or Lagrange point. It provides methods to find the center of motion id, barycenter of motion id, and to check if a given celestial body is a barycenter.

Compute the geosynchronous orbit based on the given longitude and epoch. 

This function calculates the state vector of a geosynchronous orbit in body fixed frame of the current celestial body.


\begin{DoxyParams}{Parameters}
{\em longitude} & The desired longitude at which the geosynchronous orbit should be established. \\
\hline
{\em epoch} & The epoch of the orbit, expressed in TDB (Barycentric Dynamic \mbox{\hyperlink{namespaceIO_1_1Astrodynamics_1_1Time}{Time}}). \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The calculated parameters of the geosynchronous orbit.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
The longitude is assumed to be in radian and the epoch is assumed to be in TDB.
\end{DoxyNote}
\begin{DoxyRemark}{Remarks}
The geosynchronous orbit is a circular, geocentric orbit with an inclination of zero and an argument of perigee that aligns the longitude with the position of the satellite. Therefore, all other orbital parameters are computed based on these constraints. 
\end{DoxyRemark}
